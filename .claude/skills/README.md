# Skills

This directory contains the project travel-planning skills.

- `travel-guide`: main orchestration skill for personalized travel guides.
- `travel-intake`: preference and hard-boundary collection.
- `travel-research`: Xiaohongshu, Amap, and direct `flyai-cli` research, with detailed templates in `references/`.
- `travel-publisher`: Notion page generation and final validation.

Keep each `SKILL.md` concise. Put only stable, reusable checklists in one-level reference files.
