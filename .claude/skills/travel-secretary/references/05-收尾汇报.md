## 7. 常见问题

### Q: 用户想要导出 PDF 怎么办？
A: Notion 支持导出 PDF，在页面右上角 "..." → "Export" → "PDF"。

### Q: 如何分享给朋友？
A: Notion 页面右上角 "Share" → 复制链接，设置为 "Anyone with the link"。

### Q: 攻略生成后发现数据错误？
A: 使用 Notion API 的 PATCH 方法更新错误的 block，或在 Notion UI 中手动修改。

### Q: 如何添加更多图片？
A: 在 Notion UI 中手动添加，或使用 API 追加 image_block。

## 9. 错误恢复

### 9.1 若 Notion 创建失败

1. 检查 Token 是否有效
2. 检查 Database ID 是否正确
3. 检查 blocks 数量是否超过 100
4. 查看详细错误信息，针对性修复

### 9.2 若档案更新失败

1. 检查文件路径是否正确
2. 检查文件权限
3. 手动创建缺失的目录

### 9.3 若 Memory 更新失败

1. 检查 memory 文件路径
2. 使用 Write 工具重新写入
3. 验证 JSON 格式正确性
