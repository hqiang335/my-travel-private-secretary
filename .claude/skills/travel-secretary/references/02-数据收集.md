# 第二阶段：多源数据收集

## 数据源概览

| 数据源 | 用途 | 调用方式 |
|--------|------|---------|
| xiaohongshu-mcp | 真实评价、图片、避坑经验 | MCP tool calls |
| amap-maps | 天气、距离、POI 搜索 | MCP tool calls |
| flyai CLI | 酒店/机票/门票搜索和预订链接 | Bash 命令 |

## 1. 小红书数据收集（xiaohongshu-mcp）

### 必须执行的搜索（并行进行）

#### 1.1 景点攻略搜索

```python
# 使用 xiaohongshu-mcp 的 search_feeds 工具
{
  "keyword": "{目的地} 旅游攻略 2026",
  "filters": {
    "sort_by": "最多点赞",
    "note_type": "图文"
  }
}
```

收集前 5 篇笔记的 `feed_id` 和 `xsec_token`。

#### 1.2 住宿推荐搜索

```python
{
  "keyword": "{目的地} {住宿类型} 推荐",
  "filters": {
    "sort_by": "最多收藏"
  }
}
```

收集前 3 篇笔记。

#### 1.3 美食搜索

```python
{
  "keyword": "{目的地} 必吃美食 避坑",
  "filters": {
    "sort_by": "最多点赞"
  }
}
```

收集前 5 篇笔记。

#### 1.4 避坑攻略搜索

```python
{
  "keyword": "{目的地} 旅游避坑 踩坑",
  "filters": {
    "sort_by": "综合"
  }
}
```

收集前 3 篇笔记。

### 获取笔记详情

对重要笔记调用 `get_feed_detail`：

```python
{
  "feed_id": "从搜索结果获取",
  "xsec_token": "从搜索结果获取",
  "load_all_comments": false  # 默认只加载前10条评论
}
```

### 从小红书数据中提取

- **推荐景点列表**：含评价、图片 URL
- **推荐餐厅/美食**：含评价、图片 URL
- **住宿推荐**：含图片 URL
- **避坑经验总结**：整理成要点
- **特色体验项目**：非遗、网红打卡等
- **帖子原始链接**：用于攻略中的"参考来源"

### 图片处理

小红书图片 URL 格式：
```
http://sns-webpic-qc.xhscdn.com/202604241658/495261dfdac89627a5a8e95928e94f30/notes_pre_post/1040g3k831sfv97et627g4067n1slmd249ef4hh0!nd_dft_wlteh_webp_3
```

直接使用 URL，无需下载。

## 2. 高德地图数据收集（amap-maps MCP）

### 2.1 天气查询

```python
# 使用 maps_weather 工具
{
  "city": "{目的地城市名或 adcode}"
}
```

返回：当前天气、温度、未来几天预报。

### 2.2 核心景点坐标查询

```python
# 使用 maps_geo 工具
{
  "address": "{景点名称}",
  "city": "{目的地城市}"
}
```

返回：经纬度坐标。

### 2.3 景点间距离计算

```python
# 使用 maps_distance 工具
{
  "origins": "经度1,纬度1|经度2,纬度2",  # 可以多个起点
  "destination": "经度,纬度",
  "type": "1"  # 1=驾车距离, 0=直线距离, 3=步行距离
}
```

返回：距离（米）、预计时间。

### 2.4 POI 周边搜索

```python
# 使用 maps_around_search 工具
{
  "location": "经度,纬度",
  "keywords": "餐厅",  # 或 "便利店", "停车场" 等
  "radius": "1000"  # 搜索半径（米）
}
```

返回：周边 POI 列表。

## 3. 飞猪数据收集（flyai CLI）

### ⚠️ 重要：参数名称必须精确

flyai CLI 对参数名称非常严格，必须使用正确的参数名。

### 3.1 酒店搜索

```bash
flyai search-hotel \
  --dest-name "{目的地城市}" \
  --hotel-types "{住宿类型}" \
  --check-in-date "{入住日期 YYYY-MM-DD}" \
  --check-out-date "{退房日期 YYYY-MM-DD}"
```

**正确参数**：
- `--dest-name`（不是 `--city`）
- `--hotel-types`（不是 `--type`）
- `--check-in-date` 和 `--check-out-date`（不是 `--checkin`/`--checkout`）

**返回数据结构**：
```json
{
  "data": {
    "itemList": [
      {
        "name": "酒店名称",
        "price": "价格",
        "mainPic": "图片URL",
        "detailUrl": "飞猪详情页URL",
        "address": "地址",
        "interestsPoi": "附近景点"
      }
    ]
  }
}
```

### 3.2 景点门票搜索

```bash
flyai search-poi \
  --city-name "{目的地城市}" \
  --keyword "{景点名称} 门票"
```

**重要**：
- `--city-name` 和 `--keyword` 必须分开
- 不能把城市名放在 keyword 里

**返回数据结构**：
```json
{
  "data": {
    "itemList": [
      {
        "name": "景点名称",
        "price": "门票价格",
        "mainPic": "图片URL",
        "detailUrl": "飞猪购票链接"
      }
    ]
  }
}
```

### 3.3 航班搜索

```bash
flyai search-flight \
  --origin "{出发城市}" \
  --destination "{目的地城市}" \
  --dep-date "{出发日期 YYYY-MM-DD}"
```

**正确参数**：
- `--origin`（不是 `--from`）
- `--destination`（不是 `--to`）
- `--dep-date`（不是 `--date`）

**返回数据结构**：
```json
{
  "data": {
    "itemList": [
      {
        "journeys": [
          {
            "segments": [
              {
                "depTime": "19:40",
                "arrTime": "22:10",
                "depAirport": "广州白云",
                "arrAirport": "成都天府",
                "airline": "",  // ⚠️ 可能为空
                "flightNo": ""  // ⚠️ 可能为空
              }
            ]
          }
        ],
        "jumpUrl": "https://a.feizhu.com/2576Tc"  // ✅ 飞猪短链接可用
      }
    ]
  }
}
```

**⚠️ 注意**：
- `airline` 和 `flightNo` 字段可能为空字符串
- 使用 `jumpUrl`（飞猪短链接）作为预订链接

### 3.4 高铁搜索

```bash
flyai search-train \
  --origin "{出发城市}" \
  --destination "{目的地城市}" \
  --dep-date "{出发日期 YYYY-MM-DD}"
```

## 4. 并行数据收集策略

### 并行执行示例

```python
# 同时发起多个 xiaohongshu 搜索
search_feeds(keyword="成都 旅游攻略 2026")
search_feeds(keyword="成都 精品民宿 推荐")
search_feeds(keyword="成都 必吃美食 避坑")
search_feeds(keyword="成都 旅游避坑 踩坑")

# 同时发起高德地图查询
maps_weather(city="成都")
maps_geo(address="宽窄巷子", city="成都")
maps_geo(address="大熊猫基地", city="成都")

# 同时发起 flyai 搜索（使用 Bash 并行）
flyai search-hotel --dest-name "成都" --check-in-date "2026-05-01" --check-out-date "2026-05-03" &
flyai search-poi --city-name "成都" --keyword "宽窄巷子 门票" &
flyai search-flight --origin "广州" --destination "成都" --dep-date "2026-05-01" &
wait
```

## 6. 常见问题

### Q: flyai 命令返回空结果怎么办？
A: 检查参数名称是否正确，特别是 `--dest-name`, `--origin`, `--dep-date` 等。

### Q: 小红书图片无法加载？
A: 小红书图片 URL 有时效性，建议在生成攻略时立即使用。

### Q: 高德地图返回多个同名景点？
A: 使用 `city` 参数限定城市范围，或选择评分最高的结果。

### Q: 飞猪链接是短链接，会过期吗？
A: 飞猪短链接（`a.feizhu.com`）通常长期有效，但价格以实时为准。在攻略中标注"价格仅供参考"。
