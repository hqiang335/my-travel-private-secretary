# 第三阶段：行程规划

## 核心原则

1. **地理聚类**：同一区域的景点安排在同一天，减少折返
2. **节奏匹配**：根据用户选择的节奏分配景点数量
3. **时间优化**：考虑开放时间、人流高峰、用餐时间
4. **预算控制**：在用户预算范围内优化体验

---

## 1. 景点聚类与日程分配

### 1.1 基于高德地图距离数据聚类

使用高德地图的 `maps_distance` 工具计算景点间距离，将距离近的景点归为一组。

**聚类规则**：
- 步行距离 < 2km：可安排在同一时段
- 驾车距离 < 10km：可安排在同一天
- 驾车距离 > 30km：建议单独安排一天

**示例（成都）**：
- **Day 1 老城区**：宽窄巷子 → 文殊院 → 锦里（步行/短途打车）
- **Day 2 郊区**：都江堰 → 青城山（地铁+大巴，单程1.5h）
- **Day 3 市区**：大熊猫基地 → 成都博物馆（地铁30min）

### 1.2 按用户节奏分配景点数量

| 节奏类型 | 每天景点数 | 特点 |
|---------|-----------|------|
| 特种兵型 | 4-5 个 | 紧凑，早出晚归 |
| 适中型 | 2-3 个 | 舒适，有休息时间 |
| 度假型 | 1 个 | 慢节奏，深度体验 |

### 1.3 考虑开放时间和人流

- **早晨景点**：大熊猫基地（7:30 最佳，熊猫活跃）
- **避开高峰**：热门景点工作日游览
- **夜间活动**：夜市、古街灯光秀

## 3. 餐饮安排原则

### 3.1 餐厅选择

- **午餐**：景点附近，快速方便
- **晚餐**：特色餐厅，深度体验
- **小吃**：穿插在行程中，灵活安排

### 3.2 预算分配

| 预算类型 | 早餐 | 午餐 | 晚餐 | 小吃 | 日均 |
|---------|------|------|------|------|------|
| 经济型 | ¥10-20 | ¥30-50 | ¥50-80 | ¥20 | ¥110-170 |
| 舒适型 | ¥20-40 | ¥60-100 | ¥100-200 | ¥40 | ¥220-380 |
| 豪华型 | ¥50+ | ¥150+ | ¥300+ | ¥50+ | ¥550+ |

### 3.3 从小红书整理美食推荐

- 本地人推荐（非网红店）
- 招牌菜、人均消费
- 营业时间、是否需要排队
- 避坑提示（如景区内餐厅贵且一般）

## 5. 预算汇总

### 5.1 预算表格模板

```markdown
| 项目 | 估算金额（人均） | 备注 |
|------|----------------|------|
| 大交通（往返） | ¥XXX | 飞机/高铁 |
| 住宿（N晚） | ¥XXX | 酒店均价×天数 |
| 景点门票 | ¥XXX | 各景点门票合计 |
| 餐饮（N天） | ¥XXX | 人均×天数×3餐 |
| 市内交通 | ¥XXX | 地铁+打车 |
| 其他（购物等） | ¥XXX | 弹性支出 |
| **合计（人均）** | **¥XXX** | **含往返交通** |
```

### 5.2 预算计算公式

```python
# 大交通
往返机票 = 单程价格 × 2

# 住宿
住宿总费用 = 酒店均价 × (天数 - 1)

# 门票
门票总费用 = sum(各景点门票价格)

# 餐饮
餐饮总费用 = 人均日餐饮预算 × 天数

# 市内交通
市内交通 = 天数 × 30-50元（地铁+打车）

# 其他
其他费用 = 总预算 × 10-15%（弹性）

# 人均总预算
人均总预算 = (往返机票 + 住宿 + 门票 + 餐饮 + 市内交通 + 其他) / 人数
```

## 6. 特殊场景处理

### 6.1 带娃家庭

- 每天安排 1-2 个景点
- 选择亲子友好景点（动物园、博物馆）
- 午休时间（13:00-15:00）
- 避免长时间步行

### 6.2 老人出行

- 节奏放慢，每天 1-2 个景点
- 选择无障碍景点
- 安排充足休息时间
- 避免爬山、长途跋涉

### 6.3 情侣/蜜月

- 浪漫景点（夜景、湖泊）
- 精致餐厅
- 特色民宿
- 拍照打卡点

### 6.4 单人旅行

- 安全第一（避免夜间独行）
- 青旅/社交型住宿
- 灵活行程
- 结伴游景点

## 8. 常见问题

### Q: 如何处理景点闭馆日？
A: 查询景点官网，避开周一闭馆（博物馆常见）。

### Q: 雨天怎么办？
A: 提供室内备选方案（博物馆、商场、美食街）。

### Q: 行程太紧怎么办？
A: 删减次要景点，保留核心景点，增加休息时间。

### Q: 预算超了怎么办？
A: 调整住宿档次、减少景点门票、选择经济型餐厅。
